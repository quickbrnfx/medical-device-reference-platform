# Decision Log

This document provides a chronological index of significant engineering decisions made throughout the project.

Detailed rationale is documented in the corresponding Architecture Decision Records (ADRs) or engineering artifacts.

The purpose of this log is discoverability rather than documentation.

| Date | Decision | Reference |
|------|----------|-----------|
| July 2026 | Selected the Connected Physiological Monitor as the Version 1 reference product. | ADR-001 |

## Usage

New decisions should be added in chronological order.

Each entry should contain:

- Date
- Brief decision summary
- Reference to the supporting ADR or engineering artifact

This document should not duplicate the contents of an ADR.
