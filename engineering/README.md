# Engineering

## Overview

The Engineering section contains reusable engineering knowledge that can be applied across multiple medical device projects.

Rather than documenting a single product, these resources describe the engineering methods, principles, and practices used to develop modern medical devices.

This material serves as the foundation for the reference platforms contained elsewhere in the repository.

---

## Purpose

Medical devices vary significantly in function and implementation, but many engineering activities remain consistent across products.

This section captures those reusable activities so they can be applied to different device types without redefining the engineering process for each project.

Examples include:

- Product Definition
- Requirements Engineering
- System Architecture
- Quality Engineering
- Verification & Validation
- Human Factors Engineering
- Medical Device Cybersecurity

---

## Engineering Domains

| Domain | Purpose |
|---------|---------|
| Product | Define the problem to be solved. |
| Requirements | Specify what the system must do. |
| Architecture | Organize system responsibilities into a coherent solution. |
| Quality | Address safety, cybersecurity, human factors, and other cross-cutting engineering concerns. |
| Verification | Demonstrate that the system satisfies its requirements. |

---

## How to Use This Section

Each engineering domain introduces a reusable methodology, explains its purpose, and provides guidance for creating high-quality engineering artifacts.

These documents are intended to be independent of any specific medical device implementation.

---

## Relationship to Reference Platforms

Reference platforms demonstrate how the engineering methods described in this section are applied to complete medical device systems.

Each platform uses the Engineering section as its foundation while adapting the guidance to the needs of a specific product.

The first reference platform in this repository is the **Connected Physiological Monitor**.

Future platforms will reuse these engineering methods while demonstrating their application to different classes of medical devices.
